# RS-images-prcoccess
用于遥感图像处理，功能包括灰度直方图，NDVI，图像卷积运算，图像腐蚀与膨胀运算的一些脚本
——————————————————————————————————————————————————————————————————————
新增 光谱曲线交互可视化分析，使用Landsat9数据，取其中7个波段，环境需要rasterio等（与之前的脚本独立）。
